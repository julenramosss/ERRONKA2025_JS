# API erreferentzia

Backend route handler-etako oinarri-aurrizkia: `/api/*`.

> [!NOTE]
> Atal hau backend `route.ts` kudeatzaileetatik eta frontend API bezeroengandik sortzen da. Frontend mota-sistemetan espresuki adierazita ez dauden payload xehetasunak **Egiaztapena behar du** gisa markatuta daude.

## Autentifikazio endpoint-ak

| Metodoa | Ibilbidea                  | Helburua                                                           | Autentifikazioa              | Frontend erabilera                        |
| ------- | -------------------------- | ------------------------------------------------------------------ | ---------------------------- | ----------------------------------------- |
| POST    | `/api/auth/login`          | Autentifikatu eta sarbide+freskatze tokenak eman                   | Publikoa                     | `auth-api.login`                          |
| POST    | `/api/auth/refresh`        | Freskatze-tokena biratu eta sarbide-token berria eman              | Freskatze-cookiea            | `auth-api.refresh`, Axios interceptora    |
| POST    | `/api/auth/logout`         | Freskatze-tokena errebokatu eta cookiea garbitu                    | Bearer                       | `auth-api.logout`                         |
| GET     | `/api/auth/me`             | Uneko autentifikatutako erabiltzailearen profila                   | Bearer                       | `auth-api.getMe`                          |
| POST    | `/api/auth/forgotPassword` | Berrezartzeko fluxuaren emaila sortu                               | Publikoa                     | `auth-api.forgotPassword`                 |
| PATCH   | `/api/auth/changePwd`      | Berrezartzeko/aktibatzeko token-fluxua balioztatzen edo osatzen du | Token-oinarritutako publikoa | Frontend aplikazioan zuzenean idatzita ez |

## Erabiltzaile endpoint-ak

| Metodoa | Ibilbidea                | Helburua                                          | Autentifikazioa |
| ------- | ------------------------ | ------------------------------------------------- | --------------- |
| POST    | `/api/users/create`      | Erabiltzailea sortu eta aktibatze-fluxua abiarazi | Admin           |
| GET     | `/api/users/list`        | Orrikaturiko erabiltzaile-zerrenda                | Admin           |
| GET     | `/api/users/getById?id=` | Erabiltzailearen xehetasunak id-aren arabera      | Admin           |
| PATCH   | `/api/users/update`      | Erabiltzailea eguneratu                           | Admin           |
| DELETE  | `/api/users/remove?id=`  | Erabiltzailea ezabatu (murrizketekin)             | Admin           |
| PATCH   | `/api/users/changeMyPwd` | Norbere pasahitza aldatu                          | Bearer          |

## Ibilbide eta geraleku endpoint-ak

| Metodoa | Ibilbidea                      | Helburua                                              | Autentifikazioa              | Frontend erabilera               |
| ------- | ------------------------------ | ----------------------------------------------------- | ---------------------------- | -------------------------------- |
| POST    | `/api/routes/create`           | Banatzaileentzako/datarako ibilbide optimizatua sortu | Admin                        | Oraindik frontend-era lotuta ez  |
| GET     | `/api/routes/getByUserAndDate` | Admin-eko ibilbide-bilaketa                           | Admin                        | Oraindik frontend-era lotuta ez  |
| GET     | `/api/routes/getMyDaily`       | Banatzailearen eguneroko ibilbidea                    | Banatzailea                  | `routes-api.getMyDailyRoute`     |
| GET     | `/api/routes/continueFromPast` | Iraganeko ibilbide pendientea bilatu                  | Banatzailea                  | `routes-api.getPendingPastRoute` |
| POST    | `/api/routes/continueFromPast` | Iraganeko ibilbidea gaurko egunean jarraitu           | Banatzailea                  | `routes-api.continueFromPast`    |
