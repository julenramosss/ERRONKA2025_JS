# pakAG — Documentación

Bienvenido a la documentación técnica de **pakAG**, el sistema de gestión y seguimiento de paquetes de Tolosa.

## Secciones

- **[Backend API](/backend)** — Endpoints REST, DTOs, servicios y repositorios (Next.js 15 + TypeScript + MySQL)
- **[Frontend App](/frontend)** — Aplicación React con Next.js App Router, React Query y Tailwind v4
- **[Arquitectura](/architecture)** — Estructura del monorepo, flujos y decisiones de diseño
- **[Base de datos](/database)** — Schema SQL, tablas y relaciones

## Stack

| Capa            | Tecnología                                            |
| --------------- | ----------------------------------------------------- |
| Backend         | Next.js 15, TypeScript, MySQL2, JWT, Nodemailer       |
| Frontend        | Next.js 16, React 19, TanStack Query, Tailwind CSS v4 |
| Docs            | Nextra 4, Next.js 16                                  |
| Package manager | pnpm (workspace monorepo)                             |
